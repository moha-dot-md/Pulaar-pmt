# Pulaar_pmt 

### Pmt stands for processing model and translation unit.

Well this project aims to digitalize the pulaar language in order to preserve it as much as possible.
It will be done in three steps, 

## First step : 
    Gathering data from pdf books using an intelligent program that handles pulaar characters.

## Step two : 
    Creating a model that produces text from pseudo-scratch

## Final step: 
    Creating a model that understands voice and gives transcript and produces voic.


## FUN FACT:
Ndiyam loonde is a Pulaar expression meaning “water from a clay water pot.
This name was chosen to reflect a unique identity rooted in the culture and language of its speakers, drawing inspiration from everyday life and tradition.
